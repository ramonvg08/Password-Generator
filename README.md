# 🔐 Password Generator 🔐

¡Bienvenido al **Generador de Contraseñas Seguras en Python**! 🚀  
Este proyecto te permite crear contraseñas aleatorias, fuertes y totalmente personalizables que cumplen con las mejores prácticas de seguridad. 🔒✨

---

## ⚙️ Características

✅ Genera contraseñas seguras y aleatorias.  
✅ Personaliza la longitud y los requisitos mínimos.  
✅ Incluye números, caracteres especiales, mayúsculas y minúsculas.  
✅ Utiliza el módulo `secrets` para máxima seguridad criptográfica.  
✅ Valida que todas las condiciones de seguridad se cumplan.

---

## 🐍 Requisitos

- Python 3.6 o superior

---

## 🎯 Ejemplo de salida:

Generated password: 8#nY5V!c9@dLpW2z

## 🛠️ Personalización
Puedes ajustar la configuración al llamar a la función generate_password() para que la contraseña se adapte a tus necesidades:

generate_password(length=20, nums=3, special_chars=2, uppercase=2, lowercase=2)
